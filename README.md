# Rucky
A simple rubber ducky attack apk

[Download](https://github.com/mayankmetha/Rucky/releases/latest)

## REQUIREMENTS
1) Android KitKat onwards
2) Supported Architecture

## ARCHITECTURE
- armeabi-v7a
- arm64-v8a
- x86
- x86_64
